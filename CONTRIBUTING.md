# Contributing to Dog Studio

Thank you for your interest in contributing to Dog Studio! We welcome contributions from the community.

## Code of Conduct

This project follows a code of conduct to ensure a welcoming environment for all contributors. By participating, you agree to abide by its terms.

## How to Contribute

### Reporting Bugs

If you find a bug, please open an issue on GitHub with:
- A clear title and description
- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshots if applicable
- Your environment (OS, browser, Node.js version)

### Suggesting Features

Feature requests are welcome! Please open an issue with:
- A clear description of the proposed feature
- Why it would be useful
- Any relevant examples or mockups

### Pull Requests

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature-name`
3. Make your changes
4. Ensure all tests pass and linting is clean: `pnpm lint`
5. Commit your changes with a clear commit message
6. Push to your fork
7. Open a Pull Request with a detailed description

### Development Setup

See the [Installation](#installation) section in the README for setup instructions.

### Coding Standards

- Follow the existing code style
- Use meaningful variable and function names
- Add comments for complex logic
- Ensure code is well-tested
- Run `pnpm lint` before committing

### Commit Messages

Use clear, descriptive commit messages:
- `feat: add new animation feature`
- `fix: resolve scroll trigger bug`
- `docs: update README installation instructions`

### Testing

- Test your changes thoroughly
- Ensure the app builds successfully: `pnpm build`
- Test on different browsers and screen sizes

## Project Structure

See the README for an overview of the project structure. When adding new features:
- Place components in `src/components/`
- Add styles to appropriate CSS files
- Update the README if necessary

## Questions?

If you have questions about contributing, feel free to open an issue or contact the maintainers.

Thank you for contributing to Dog Studio! 🎉
